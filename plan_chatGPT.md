# 1. СТАРТ — Составьте свой личный план с ИИ

Первое, что важно — понять свою:

- Точку А — где вы сейчас
- Точку Б — куда хотите прийти
- Срок — за какое время

## Промпт для ChatGPT:

```
Точка А: Я полностью новичок в испанском языке.

Точка Б: Хочу говорить на уровне выше базового с местными в Испании за 6 месяцев: на рынке, в кафе, с соседями, у врача.

Время: Я могу заниматься 1 час в день 5 раз в неделю Составь мне детальный план обучения по принципу Парето.

Распиши по неделям на 6 месяцев. Когда я отправлю тебе номер недели — распиши по дням, добавь домашние задания и бесплатные ресурсы.
```

### Готово! У вас на руках — персональный учебный маршрут на 6 месяцев.

Закрепите чат и возвращайтесь к нему каждую неделю.

# 2. ПОГРУЖЕНИЕ — Загружаем испанский в мозг

## Частотная лексика

```
Составь список из 300 самых употребляемых слов в испанском языке (существительные, прилагательные, глаголы, наречия).

Начинай с самых частых.

К существительным — артикль и множественное число.

К глаголам — формы настоящего, прошедшего времени и причастие.

Пример предложения для уровня А1 и перевод — на новой строке.
```

## Грамматическая тема

```
Я учу испанский с нуля (или указать уровень).

Объясни тему и составь для меня 20 предложений с пропущенными словами, чтобы я могла потренировать [ser/estar, прошедшее время, притяжательные местоимения и т.д.]. 

Добавь ключ с объяснением.
```

## По фото из учебника

```
Вот фото страницы из учебника (прикрепить фото). Составь для меня дополнительные упражнения по этой теме.
```

# 3. ПРАКТИКА — Говорим, читаем, играем

## Истории со словарём

```
Напиши короткую историю с этими словами: [список] для уровня A1.

Выдели новые слова курсивом.

Добавь вопросы, грамматический квиз и пересказ.
```

## Мой день

```
Хочу рассказать про свой день.

Помоги, задавай уточняющие вопросы, чтобы я могла попрактиковать настоящее время / прошедшее / [список слов].
```

## Диалоги и ролевые игры

```
Смоделируй диалог в кафе.

Ты — официант, я — клиент. Уровень A1.

Вовлекай меня в разговор.
```

## Обратная связь по письму

```
Вот мой текст.

Исправь ошибки, предложи более естественные фразы и дай упражнения для слабых мест.
```

## Геймификация

```
Поговори со мной как Супермен / Гарри Поттер и учи испанскому!

Придумай игру на A1: бинго или угадай слово.
```

## Аудирование

```
Расскажи короткую историю про еду на уровне A1, медленно.

Потом задай мне вопросы на понимание.
```

# 4. РАЗБОР СОМНЕНИЙ

ChatGPT помогает не только учить, но и разбираться.

## Примеры промптов:

### Объяснение сложных тем

```
Объясни разницу между por и para с примерами. Дай упражнения на закрепление.
```

### Анализ заданий

```
Вот мои ответы. Проверь, где ошибки, объясни почему так, и подбери похожие задания.
```

# 5. МОТИВАЦИЯ И КОРРЕКТИРОВКА

Важно отслеживать прогресс и корректировать маршрут.

## Промпт для самопроверки:

```
Сделай для меня тест на 20 вопросов по темам: настоящее время, ser, por/para, частотные слова. Уровень A1.   В конце — разбор ошибок и рекомендации.
```

### Делайте такой тест:

- через месяц — проверка слов и грамматики
- через 3 месяца — ролевой диалог

## Важно помнить: ChatGPT — это не живой преподаватель. Возможны ошибки:

- Миксует разные диалекты
- Не обучает произношению
- Пропускает грамматические нюансы

## Рекомендации:

- Проверяйте спорные моменты в учебниках или с преподавателем
- Слушайте живую речь (YouTube, сериалы)
- Подключайте живую практику!

Язык нужен не для переписки с ИИ, а для живого общения!

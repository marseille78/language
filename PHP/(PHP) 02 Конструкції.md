# Конструкції

if (bool) {
  code
} --- Конструкція *IF*. Якщо умова *bool* є істинною, виконується певний блок коду *code*. Якщо умова є хибною, цей блок коду пропускається.



if (bool) {
  code1
} else {
  code2
} --- Конструкція *IF-ELSE*. Якщо умова *bool* є істинною, виконується один блок коду *code1*, а якщо хибною — виконується інший *code2*.



bool ? $val1 : $val2 --- Ця конструкція називається *тернарний оператор*. Вона перевіряє умову *bool* й повертає одне значення *$val1*, якщо умова істинна, і інше *$val2* — якщо хибна.



$val1 ?? $val2 --- Ця конструкція називається *оператор злиття з null*. Вона перевіряє, чи перше значення *$val1* не є *null*, і повертає його, інакше повертає друге значення *$val2*.



if (bool1) {
  code1
} elseif (bool2) {
  code2
} else {
  code3
} --- Ця конструкція називається умовний оператор *if-elseif-else*. Вона перевіряє першу умову *bool1*, і якщо вона істинна, виконує відповідний код *code1*. Якщо перша умова хибна, перевіряється друга *bool2* і якщо вона істинна, виконується блок коду *code2*. Якщо жодна з умов не істинна, виконується останній блок коду *code3*.



if (bool1):
  code1
elseif (bool2):
  code2
else:
  code3
endif; --- Ця *альтернативна конструкція* називається умовний оператор *if-elseif-else*. Вона перевіряє першу умову *bool1*, і якщо вона істинна, виконує відповідний код *code1*. Якщо перша умова хибна, перевіряється друга *bool2* і якщо вона істинна, виконується блок коду *code2*. Якщо жодна з умов не істинна, виконується останній блок коду *code3*.



switch ($x) {
  case $a:
    code1;
    break;
  case $b:
    code2;
    break;
  default:
    code3;
} --- Конструкція *switch* перевіряє значення змінної *$x* і порівнює його з кількома можливими значеннями *$a*, *$b* тощо.  

1. Спочатку *$x* порівнюється зі значенням *$a*. Якщо вони рівні, виконується *code1*, після чого *break* завершує виконання конструкції.  
2. Якщо *$x* не дорівнює *$a*, перевіряється порівняння з *$b*. Якщо значення збігається, виконується *code2*, після чого знову *break* припиняє подальше виконання.  
3. Якщо *$x* не збігається ні з *$a*, ні з *$b*, виконується *code3*, який знаходиться в блоці *default*.  

Конструкція використовується, коли потрібно порівняти одну змінну *$x* з кількома варіантами значень (*$a*, *$b* тощо) і виконати відповідний код.



switch ($x):
  case $a:
    code1;
    break;
  case $b:
    code2;
    break;
  default:
    code3;
endswitch; --- *Альтернативна конструкція switch* перевіряє значення змінної *$x* і порівнює його з кількома можливими значеннями *$a*, *$b* тощо.  

1. Спочатку *$x* порівнюється зі значенням *$a*. Якщо вони рівні, виконується *code1*, після чого *break* завершує виконання конструкції.  
2. Якщо *$x* не дорівнює *$a*, перевіряється порівняння з *$b*. Якщо значення збігається, виконується *code2*, після чого знову *break* припиняє подальше виконання.  
3. Якщо *$x* не збігається ні з *$a*, ні з *$b*, виконується *code3*, який знаходиться в блоці *default*.  

Конструкція використовується, коли потрібно порівняти одну змінну *$x* з кількома варіантами значень (*$a*, *$b* тощо) і виконати відповідний код.



for ($i = 1; $i < $num; $i++) {
  code
} --- Цикл *FOR* виконує блок коду *code* кілька разів, починаючи з *$i = 1* і збільшуючи число на 1 при кожному повторенні, доки воно менше значення *$num*. Функція виконує блок коду *code*



for ($i = 1; $i < $num; $i++):
  code
endfor; --- Цикл *FOR (альтернативний запис)* виконує блок коду *code* кілька разів, починаючи з *$i = 1* і збільшуючи число на 1 при кожному повторенні, доки воно менше значення *$num*. Функція виконує блок коду *code*



while ($i < $num) {
  code
} --- Цикл *WHILE* виконує блок коду *code* багато разів, доки значення істинне значення *$bool*. Перед кожним виконанням перевіряється умова, і якщо вона залишається істинною, код виконується знову.



while ($i < $num):
  code
endwhile; --- Цикл *WHILE* (альтернативний запис) виконує блок коду *code* багато разів, доки значення *$i* менше *$num*. Перед кожним виконанням перевіряється умова, і якщо вона залишається істинною, код виконується знову.



do {
  code
} while ($i < $num); --- Цикл *DO-WHILE*. Цей запис гарантує, що блок коду *code* виконається хоча б один раз, а потім буде повторюватися, поки значення *$i* менше *$num*. Умова перевіряється після першого виконання коду.



break --- Ця комадна дозволяє *достроково зупинити виконання циклу*, навіть якщо його умова ще виконується. Код після циклу продовжить працювати далі.



continue --- Ця команда *припиняє поточну ітерацію циклу* та переходить до наступної, не виконуючи код, який іде після неї в межах цієї ітерації.



break 2 --- Ця команда *припиняє виконання не одного*, а одразу *двох* вкладених циклів, у яких вона знаходиться, і продовжує виконання коду після них.



foreach ($arr as $val) {
  code
} --- Цикл *FOREACH* перебирає всі елементи вказаного списку (масиву) *$arr* по черзі, зберігаючи кожен елемент у змінну *$val*, після чого виконує заданий код *code*.



foreach ($arr as $key => $val) {
  code
} --- Цикл *FOREACH* перебирає всі елементи вказаного списку (асоціативного масиву) *$arr* по черзі, де кожен елемент має ключ *$key* і значення *$val*, після чого виконує заданий код *code*.



foreach ($arr as $val):
  code
endforeach; --- Цикл *FOREACH* (альтернативний запис) перебирає всі елементи вказаного списку (масиву) *$arr* по черзі, зберігаючи кожен елемент у змінну *$val*, після чого виконує заданий код *code*.



declare(strict_types=1) --- Ця конструкція вмикає *строгу типізацію* в PHP. Вона змушує PHP перевіряти типи даних під час виконання коду, і якщо значення не відповідає очікуваному типу, то виникає помилка. Це допомагає уникати неявних перетворень типів і робить код більш передбачуваним і безпечним.



include "file.php";
include("file.php") --- Конструкція в PHP, яка підключає та виконує код із зовнішнього файлу *file.php* в тому місці, де вона викликається. Якщо файл не знайдено, PHP виведе попередження (warning), але скрипт продовжить виконання.



include_once "file.php";
include_once("file.php") --- Конструкція в PHP, яка підключає та виконує код із зовнішнього файлу *file.php* *лише один раз*, навіть якщо вона викликається кілька разів. Якщо файл вже був підключений раніше, повторне підключення *не відбудеться*. Якщо файл не знайдено — буде попередження, але скрипт продовжить виконання.



require "file.php";
require("file.php") --- Конструкція в PHP, яка підключає та виконує код із зовнішнього файлу *file.php*. Якщо файл не знайдено або містить помилки, PHP *зупинить виконання скрипта* і виведе *фатальну помилку*. На відміну від *include* гарантує, що файл обов’язково має бути присутнім для подальшої роботи скрипта.



require_once "file.php";
require_once("file.php") --- Конструкція в PHP, яка підключає файл *file.php* *лише один раз*, навіть якщо виклик повторюється. Якщо файл уже був підключений раніше, повторне підключення не відбудеться. Якщо файл не знайдено, PHP зупиняє виконання скрипта та виводить *фатальну помилку*. Ця конструкція використовується для уникнення *повторного завантаження одного й того ж файлу*, наприклад, з функціями або класами.

# (PHP) 10 Функції

## Конструкції

function foo($x = 0) {
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo* з параметром *$x*, який за замовчуванням має значення *0*. Якщо аргумент не переданий під час виклику, буде використано значення *0*. Функція виконує блок коду *code*



function foo(&$x) {
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo* з параметром *$x*, переданим *за посиланням*. Це означає, що зміни змінної *$x* всередині функції впливають на її значення за межами функції. Функція виконує блок коду *code*



function foo() {
  global $x;
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo*, у якій змінна *$x* оголошується як *глобальна* за допомогою *ключового слова*. Це дозволяє функції використовувати змінну *$x*, яка була визначена за межами функції, змінювати її значення та зберігати ці зміни поза функцією. Після цього функція виконує блок коду *code*



function foo() {
  $GLOBALS[$x];
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo*, яка використовує *суперзмінну* для доступу до глобальних змінних. Це дозволяє звертатися до змінної з глобальної області видимості, де ім'я змінної визначено в *$x*. Після цього функція виконує блок коду *code*.



function foo() {
  static $x = 0;
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo*, в якій змінна *$x* оголошена як *статична* і дорівнює *0*. Це означає, що значення змінної *$x* зберігається між викликами функції, і вона не ініціалізується заново при кожному виклику (на відміну від звичайних змінних, які ініціалізуються заново при кожному виклику функції). Після цього функція виконує блок коду *code*.



return --- (Конструкції)

Ключове слово яке в PHP використовується для *повернення значення* з функції або методу. Коли воно виконується, функція припиняє виконання і повертає вказане значення, яке можна використовувати в коді, що викликав функцію.



function foo(...$x) {
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo*, яка приймає *змінну кількість аргументів*. Аргументи передаються у вигляді масиву *$x*, де кожен елемент масиву відповідає окремому переданому значенню. Функція виконує блок коду *code*.



foo(...[$a, $b]) --- (Конструкції)

Ця конструкція використовує оператор розпакування для передавання елементів масиву *[$a, $b]* як окремі аргументи у функцію *foo()*. Це означає, що масив розкривається, і значення *$a* і *$b* передаються як окремі аргументи.



function foo(array $x) {
  code
} --- (Конструкції)

Ця конструкція визначає функцію *foo*, яка приймає один параметр *$x*, що обов'язково має бути масивом. Тобто, при виклику функції потрібно передати масив як аргумент для параметра *$x*. Функція виконує блок коду *code*



function foo(callable $a, $b) {
  code
} --- (Конструкції)

(PHP 5.4+) Ця конструкція визначає функцію *foo*, де перший параметр *$a* повинен бути *викликаємим*, тобто функцією, методом або об'єктом, який можна викликати. Другий параметр *$b* може бути будь-яким типом даних. Функція виконує блок коду *code*



function foo(int $i, string $s, bool $b, int ...$nums):bool {
  code
} --- (Конструкції)

(PHP 7+) Ця конструкція визначає функцію *foo*, яка приймає три параметри з фіксованими типами: 

- *$i* має бути цілим числом,

- *$s* має бути рядком,

- *$b* має бути булевим значенням.

Також є параметр *$nums*, який дозволяє передавати *змінну кількість* цілих чисел, що зберігаються в масиві. Функція повертає значення булевого типу. Функція виконує блок коду *code*



## Функції

func_num_args() --- (Функції)

Ця функція повертає кількість аргументів, переданих у функцію при її виклику. Вона дозволяє визначити, скільки параметрів було передано в функцію, навіть якщо їх кількість змінна. (Викликається внутрі функції)



func_get_arg($n) --- (Функції)

Ця функція дозволяє отримати значення аргументу, переданого в функцію, за його порядковим номером *$n*. Номер *$n* вказує на позицію аргументу, починаючи з нуля. (Викликається внутрі функції)



func_get_args() --- (Функції)

Ця функція повертає масив всіх аргументів, переданих у функцію, незалежно від їх кількості. (Викликається внутрі функції)
